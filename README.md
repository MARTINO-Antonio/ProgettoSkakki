# PROGETTO SKAKKI

Applicazione scritta in Python, con interfaccia per terminale, che simula il gioco degli scacchi con quattro modalità: Umano vs Umano, Umano vs Computer, Computer vs Umano e modalità Spettatore, in cui due motori si sfidano tra loro.

(Il modulo dell'AI è esterno, ma l'ho voluto integrare per avere la logica avversaria e rendere il progetto più completo e divertente da usare.)
